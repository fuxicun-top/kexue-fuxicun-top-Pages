# 小周CF科学设置页面 模板

> Cloudflare Workers 科学上网设置页面的静态前端模板

## 项目简介

本项目为 Cloudflare Workers 代理服务的前端页面模板，提供管理后台、登录页面等功能。

## 访问地址

- **GitHub Pages**: https://fuxicun-top.github.io/cf-pages

## 功能特性

- 管理后台页面
- 用户登录页面
- 订阅链接生成
- 节点位置信息展示

## 部署方式

本项目通过 GitHub Actions 自动部署到 GitHub Pages。

## 相关项目

- **Cloudflare Worker 后端**: 需单独部署 `worker.js` 到 Cloudflare Workers
